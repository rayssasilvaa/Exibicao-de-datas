# 📅 Sistema de Formatação de Datas (Java)

## 🇧🇷 Descrição (PT-BR)

Este é um pequeno sistema desenvolvido em Java com o objetivo de praticar a formatação de datas utilizando diferentes localidades (*Locale*).

O programa solicita ao usuário que informe um país e, com base na escolha, exibe a data atual no formato padrão daquele país.

## 💡 Funcionalidades

* Entrada de dados via terminal
* Uso da classe `Locale` para diferentes regiões
* Formatação de datas com `DateFormat`
* Manipulação de datas com `Calendar`
* Tratamento de exceções

## 🌍 Países suportados

* Brasil
* Estados Unidos
* Itália
* Chile

## 🛠️ Tecnologias utilizadas

* Java
* API de Datas (`DateFormat`, `Calendar`)
* `Locale`
* `Scanner`

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Compile o arquivo:

```bash
javac DatasENumeros.java
```

3. Execute o programa:

```bash
java DatasENumeros
```

4. Digite o nome de um país quando solicitado:

```
Escreva seu país para ver a data de hoje:
```

## 📌 Exemplo de uso

```
Entrada: brasil  
Saída: Data padrão no Brasil: sexta-feira, 27 de março de 2026
```

## 🎯 Objetivo do projeto

Este projeto foi criado com fins educacionais para praticar:

* Manipulação de datas em Java
* Internacionalização (i18n)
* Boas práticas com entrada de dados

# 📅 Date Formatting System (Java)

## 🇺🇸 Description (EN-US)

This is a small Java project created to practice date formatting using different locales.

The program asks the user to enter a country and then displays the current date formatted according to that country's standard.

## 💡 Features

* User input via terminal
* Use of `Locale` for different regions
* Date formatting with `DateFormat`
* Date handling with `Calendar`
* Exception handling

## 🌍 Supported Countries

* Brazil
* United States
* Italy
* Chile

## 🛠️ Technologies Used

* Java
* Date API (`DateFormat`, `Calendar`)
* `Locale`
* `Scanner`
